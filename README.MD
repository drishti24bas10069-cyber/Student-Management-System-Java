# Student Management System – Java

This is a simple console-based **Student Management System** made using **Core Java (OOP + File Handling + Collections)**.

The project allows **Admin** to manage Students, Courses, Enrollments, and Results.  
Students can log in using Student ID and view their details.

---

## Features

### Admin
- Add / Update / Delete Students  
- Add / Update / Delete Courses  
- Enroll Students in Courses  
- Add or Update Results  
- View All Students  
- View All Courses  
- View All Enrollments  
- View All Results  
- Data saved permanently using Serialization

### Student
- Login using Student ID  
- View Profile  
- View Enrollments  
- View Results  

---

## Technologies Used
- Java 8+  
- OOP (Classes, Objects, Inheritance, Encapsulation)  
- Collections Framework (HashMap, ArrayList)  
- Serialization (File Storage)  
- CLI (Menu-based Application)
