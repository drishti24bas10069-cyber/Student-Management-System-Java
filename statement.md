## Objective
To design and implement a basic Student Management System using Core Java that demonstrates concepts of Object-Oriented Programming, File Handling, and a Menu-driven CLI application.

## Problem Definition
Educational institutes need a simple system to manage student information, course details, enrollments, and academic results.  
This project provides a lightweight solution using Java.

## Scope of the Project
- Student CRUD Operations  
- Course CRUD Operations  
- Enrollment Management  
- Result Management  
- Admin Login  
- Student Login  
- Persistent data storage using Serialization  

## Target Users
- Students  
- Faculty  
- Admin Staff  

## Functional Requirements
1. **Admin Login**  
2. **Manage Students**  
3. **Manage Courses**  
4. **Enroll Students**  
5. **Add / Update Results**  
6. **View Enrollments & Results**  
7. **Student Login & View Details**

## Non-functional Requirements
- Simple CLI  
- Fast performance  
- Easy to use  
- No external database required  

## Tools & Technologies
- Java 8+  
- Collections Framework  
- Serialization  
- Command Line Interface
